# موقعي البسيط مع Supabase

مدونة بسيطة تعرض منشورات من قاعدة بيانات Supabase.

## التقنيات المستخدمة
- HTML5, CSS3, JavaScript
- Supabase كقاعدة بيانات
- GitHub Pages للاستضافة

## كيفية الإعداد
1. أنشئ جدول `posts` في Supabase
2. أضف بعض المنشورات التجريبية
3. غير بيانات الاتصال في ملف `script.js`
4. ارفع الملفات على GitHub وفعل Pages

## التصميم
تصميم عصري ونظيف مع دعم كامل للغة العربية.
